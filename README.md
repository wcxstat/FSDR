# FSDR with Multivariate Responses

## Overview

This repository contains the **MATLAB code and datasets** used in the paper:

> **Functional Sufficient Dimension Reduction with Multivariate Responses: A Projection Averaging Method and Beyond**

**Authors:** Wenchao Xu, Jicai Liu, and Jianjie Guo  
**Journal:** *Statistics and Computing* (2026), 36:53  
**DOI:** 10.1007/s11222-025-10814-4  

---

## Usage

Before running any MATLAB code, please execute the following command to add all required paths:

```matlab
addpath(genpath(fullfile('..')));
```
